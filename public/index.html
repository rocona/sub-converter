Surge Converter

把通用订阅转换成可直接落地的 Surge 配置

面向 vmess、ss、trojan 的通用转换器，支持对有问题的 Trojan 节点统一补全 WebSocket 参数，适合放到 GitHub 后再交给 Zeabur 持续部署。

输出目标 Surge [Proxy]
重点修复 Trojan + WebSocket
部署方式 GitHub → Zeabur

订阅输入

你可以粘贴订阅链接，或者直接贴原始订阅内容。

Trojan 修补

针对订阅没把 WebSocket 参数转换出来的情况。

转换结果

等待输入。转换后会直接生成可粘贴到 Surge 的 [Proxy] 段。
